# Match Prediction Microsite

## Setup
1. Create a Google Sheet.
2. Open Extensions > Apps Script.
3. Paste `Code.gs` into Apps Script.
4. Change `ADMIN_PASSWORD`.
5. Deploy > New deployment > Web app.
   - Execute as: Me
   - Who has access: Anyone
6. Copy the Web App URL and paste it in `config.js`.
7. Upload this folder to Netlify, Vercel, GitHub Pages, or your hosting.

## Admin
Open `/admin.html`, enter the password, edit qualified team names, save, or download CSV.
